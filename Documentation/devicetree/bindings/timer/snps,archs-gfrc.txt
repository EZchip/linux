Synopsys ARC Free Running 64-bit Global Timer for ARC HS CPUs
- clocksourc provider for SMP SoC

Required properties:

- compatible : should be "snps,archs-gfrc"
- clocks     : phandle to the source clock

Example:

	timer1: timer_clksrc {
		compatible = "snps,archs-gfrc";
		clocks = <&core_clk>;
	};
